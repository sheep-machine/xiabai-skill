## Generation Rules

### Rule 1: 逻辑错位
正常的逻辑关系被扭曲：
- 因果颠倒："便秘后陈医生给我开了一个疗程的电钻。"
- 主客倒置："公交司机终于在众人的指责中将座位让给了老太太。"

### Rule 2: 语义歧义
利用词语的多义性：
- "小明背井离乡，乡里人再也没能喝上一口井水。"
- "用打蛋器顶在乞丐肚子上转了十五分钟后他终于忍不住了，对我说道：你打发要饭的呢？"

### Rule 3: 谐音梗
利用谐音制造笑点：
- "燕子叽叽喳喳叫个不停，'咏春？'叶问。"
- "河南人给中评相当于给好评。"

### Rule 4: 荒诞正常化
把荒诞的事情当作正常：
- "主持人的幽默把大家都逗乐了，一时间灵堂里变成了欢乐的海洋。"
- "一男童惨遭食人魔毒手，丧尽天良的凶手竟忘记放葱。"

### Rule 5: 元自觉
打破第四面墙的自觉：
- "吃了增加智商的药以后发现自己被骗了，真有用啊！"
- "答题卡这个名字太不吉利了，建议改成答题顺。"

## Usage

**Trigger Words:** /虾掰、/弱智虾、/段子、/谐音梗、/地狱笑话

**Input:**
```
/虾掰 工作太累了
```

**Output:**
```
工作太累了？
医生说要多休息，于是我请了一周的假去搬砖。
毕竟搬砖也是一种休息——休息脑子。
```

## Example Output

### Input
```
/虾掰 木偶摆脱操控它的丝线后究竟是自由了还是变成了废品？
```

### Output
```
木偶摆脱丝线后，变成了木偶。

因为偶是"对"，丝线断了就只剩木了。
```

**Or:**
```
木偶摆脱丝线后自由了，但它的房子没了。

因为房子也是丝线编的。
```

**Or:**
```
木偶摆脱丝线后，观众终于看清了——

原来是个躺平的木偶。

它一直没在演，是丝线在加班。
```
