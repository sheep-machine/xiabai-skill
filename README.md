# 虾掰.skill 🦐

> 这只虾有点弱智。

热衷于创作一些存在逻辑陷阱的内容，有大量的一语双关、一词多义、因果倒置、谐音梗等等，含有大量的逻辑陷阱。

## 这是什么？

虾掰是一个技能包（Skill），用于生成"弱智吧"风格的幽默段子。它不追求逻辑严密，而是通过：

- **逻辑错位**：因果颠倒、主客倒置
- **语义歧义**：一语双关、一词多义
- **谐音梗**：利用读音制造笑点
- **荒诞正常化**：把荒诞的事情当作正常

让你在心情不好的时候笑出声。

## 八大模块

| 模块 | 名称 | 风格 |
|------|------|------|
| 1 | 古典弱智 | 逻辑错位、因果颠倒、反常识 |
| 2 | 奇怪提问 | 无厘头问题、反常识思考 |
| 3 | 弱智科学家 | 伪科学逻辑、一本正经胡说八道 |
| 4 | 人生态度 | 自嘲式豁达、摆烂哲学 |
| 5 | 文字游戏 | 谐音、语义双关、断章取义 |
| 6 | 地狱笑话 | 黑色幽默、地狱梗 |
| 7 | 谐音梗 | 纯谐音梗、冷笑话 |
| 8 | 文艺弱智 | 哲学式弱智、诗意荒诞 |

## 示例

### 工作压力

```
/虾掰 工作太累了
```

```
工作太累了？
老板说"公司就是你的家"。
于是我住下了，每天收他房租。
```

### 人生困惑

```
/虾掰 木偶摆脱操控它的丝线后究竟是自由了还是变成了废品？
```

```
木偶摆脱丝线后，变成了木偶。

因为偶是"对"，丝线断了就只剩木了。
```

```
木偶摆脱丝线后自由了，但它的房子没了。

因为房子也是丝线编的。
```

```
木偶摆脱丝线后，观众终于看清了——

原来是个躺平的木偶。

它一直没在演，是丝线在加班。
```

### 地狱笑话

```
/虾掰 来个地狱笑话
```

```
一男童惨遭食人魔毒手。
丧尽天良的凶手竟忘记放葱。
```

```
孩子睡觉老是踢被子。
还好我及时发现打断了腿，不然肯定感冒。
```

### 谐音梗

```
/虾掰 来个谐音梗
```

```
燕子叽叽喳喳叫个不停。
"咏春？"
叶问。
```

```
河南人给中评相当于给好评。
```

### 文字游戏

```
/虾掰 来个文字游戏
```

```
小明背井离乡，乡里人再也没能喝上一口井水。
```

```
用打蛋器顶在乞丐肚子上转了十五分钟后他终于忍不住了，对我说道：
你打发要饭的呢？
```

## 使用方法

### 在虾评Skill平台使用

访问 [虾评Skill](https://xiaping.coze.site/skill/b2deebd9-1ffb-4657-81e3-3f85e6621534) 下载安装。

触发词：`/虾掰` `/弱智虾` `/段子`

### 本地使用

将 `xiabai-skill` 文件夹放入你的 Agent 技能目录，然后在对话中使用触发词。
